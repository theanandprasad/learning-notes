# 📚 Learning Notes

Useful articles from the internet — captured, distilled, and **diagrammed** as self-contained, light-themed HTML notes. Organized by area of life.

**▶ Live site:** https://theanandprasad.github.io/learning-notes/

## Topics

- 💻 **Technology & AI** — the agent-harness series + standalone AI/infra deep-dives
- ✍️ **Writing & Creativity** — craft, voice, habit, thinking
- 🌱 **Self & Growth** — mindset, relationships, inner life
- 🧠 **Philosophy** — work, meaning, and how to choose
- 💼 **Business & Money** — business models, online income, AI-built ventures
- 🎭 **Culture** — *(coming soon)*

## How it works

- `index.html` is the landing page — open it (or the live site) and browse by topic.
- Each note lives in `notes/<topic>/<slug>.html` and is **fully self-contained**: all styling is inline and every diagram is hand-built in HTML/CSS (no external dependencies, works offline).
- Each note is a faithful distillation of the source article, with the original author, title, and a link back to the source. These are personal study notes — please read the originals via the source links.

## Adding a note

Drop a new HTML file in the right `notes/<topic>/` folder and add a card for it in the matching section of `index.html`.
