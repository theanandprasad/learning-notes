Making Playgrounds Using Claude Code

By Thariq (@trq212, Anthropic) • Published Jan 30, 2026 • Saved Jun 27, 2026
Claude Code Playgrounds Visual Interaction Plugins

Some things are hard to explain to a model in text — a layout, an architecture, the feel of a game. The fix is to stop typing and start looking: have Claude build you a little HTML world you can poke at, then hand its conclusions back.

The idea

Anthropic published a Claude Code plugin called playground that helps Claude generate HTML playgrounds — standalone HTML files that let you visualize a problem with Claude, interact with it, and get an output prompt to paste back into Claude Code. It's great for interacting with the model in ways that aren't well-suited to text.

The loop

A playground is a single throwaway HTML file. You don't read code to use it — you click around in a real interface, and when you're done it hands you a prompt summarizing your decisions. That prompt goes straight back into Claude Code to actually make the changes.

What it's good for

Codebase architecture
visualize how a system fits together
Component design
adjust the design of a component
Layout & design
brainstorm layout and design
Game balance
tweak the balance of a game

Install

install in Claude Code/plugin marketplace update claude-plugins-official /plugin install playground@claude-plugins-official

Example playgrounds

Each is invoked by asking Claude to use the playground skill, then describing the unique interaction you want. The prompts below are reproduced verbatim.

Redesigning the AskUserQuestion tool
Use the playground skill to create an playground that helps me explore new layout changes to the AskUserQuestion Tool
Critiquing your writing
Use the playground skill to review my SKILL.MD and give me inline suggestions I can approve, reject or comment
Tweaking a Remotion video intro
Use the playground skill to tweak my intro screen to be more interesting and delightful
Viewing an architecture diagram & commenting
Use the playground skill to show how this email agent codebase works and let me comment on particular nodes in the architecture to ask questions, make edits, etc
Balancing a roguelike game
Use the playground skill to help me balance the 'Inferno' hero's deck

The tip

Think of a unique way of interacting with the model and then ask it to express that.

The playground is a way to break out of pure-text interaction. The bottleneck usually isn't what Claude can build — it's how you tell it what you want. When the thing is visual or spatial, describing it in prose is the wrong tool. Pick the interaction that fits the problem, and have Claude express it as something you can touch.
